HTTP:
Clients and servers -
    can communicate but exchanging individual messages
    The messages sent by the client/web browser are called Request
    The messages sent by the Server are the Responses

Request:
    GET: get an html file
    POST: post something, send some data to servers and post/add to server/database
    PUT: update the data thats sent
    DELETE: delete data on the server

Response:
    Status Code: 200
